# File Upload API ([API DOC](https://cloud-upload-api.kyawzinthiha.dev))

This is a starter code for building api for uploading files using **Nest.js** and **AWS S3**

## Usage

```bash
npm run build
npm run start
```

## Required Variables

- WEB_URL
- COOKIE_SECRET
- DATABASE_URL
- AWS_REGION
- AWS_BUCKET
- AWS_S3_DOMAIN
- AWS_ACCESS_KEY_ID
- AWS_SECRET_ACCESS_KEY
- AWS_S3_FOLDER_PATH
